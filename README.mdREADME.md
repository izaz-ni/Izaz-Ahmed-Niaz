
# Izaz Ahmed Niaz — Portfolio (Starter)

This repository contains a static portfolio starter site for Izaz Ahmed Niaz.

How to publish:
1. Upload files (index.html, styles.css, script.js, assets/*).
2. In repository Settings → Pages: set Source to branch `main` and folder `/` then Save.
3. Visit the Pages URL shown in the Settings (e.g., https://izaz-ni.github.io/Izaz-Ahmed-Niaz/).

Replace placeholders:
- Edit index.html: replace your-email@example.com with your real email.
- Replace images in assets/ with your hero and project images.
- Update portfolio items in the Selected Work section.

SEO tips:
- Submit sitemap.xml to Google Search Console after site is live.
- Use descriptive alt text for images and compressed WebP when possible.
